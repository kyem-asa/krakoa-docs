# Welcome to Gardens of Krakoa
 
Gardens of Krakoa is an API for information on X-Men characters from the Marvel Universe.

Read the documentation to get started or learn how to contribute.

Want to dive in? Access everything at:  

[Gardens of Krakoa](https://gardens-of-krakoa.cyclic.app/api/v1)

## Documentation

<Callout type="info" emoji="ℹ️">
  <DocumentCount />
</Callout>


The API includes documents with these properties.


|  Property |  Type | Value  |
|---|---|---|
| name | STRING | Common name or alias (ex. 'Storm')
| fullName| STRING  | Full name (ex. 'Ororo Munroe') |
| aliases  | ARRAY  |  Known aliases (ex. 'Storm)|
| teamAffiliations   | ARRAY  | Current or former teams (ex. 'Hellfire Club'). |
| firstAppearances | STRING | First appearance in Marvel Comics (ex. 'Giant-Size X-Men #1')
| powers | ARRAY | Powers and abilities (ex. [ 'Ecological empathy',  'Flight'] )

### Endpoints

The following endpoints provide access to the data.

#### All data

```js filename="endpoints.js" {3} copy
https://xmen-api.cyclic.app/api
```

#### All characters

```js filename="endpoints.js" {3} copy
https://gardens-of-krakoa.cyclic.app/api/v1
```


#### A specific character

```js filename="endpoints.js" {3} copy
https://xmen-api.cyclic.app/character
```


Replace `character` with the character you are searching for. 

```js filename="endpoints.js" {3} copy
https://localhost:8000/api/storm
```
