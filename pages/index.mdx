# Introduction
 
The X-Men API provides information about Marvel characters who have been or currently are members of the X-Men and affiliated teams. 

Read the documentation to get started or learn how to contribute.

Want to dive in? Access all characters at:

`https://localhost:8000/api`

## Documentation

### What's in the data?

Each object is a character with the following properties:

| Property      | Value |
| ----------- | ----------- |
| name      | A string       |
| aliases   | Array of aliases the character is known by        |
| team affiliations      | Current or former teams the character has participated in       |
| first appearances   | The character's first appearance in Marvel Comics        |
| powers   | Array of character's abilities and powers        |


### Endpoints:

The following endpoints will let you access the data directly.

`https://localhost:8000/api`

To view all characters:

`https://localhost:8000/api`

To access a specific character:

`https://localhost:8000/api/character`

Replace character with the character you are searching. For example:

`https://localhost:8000/api/storm`
