# Welcome to Gardens of Krakoa
 

Gardens of Krakoa is an X-Men API providing information about Marvel characters who have been or currently are members of the X-Men and affiliated teams. 

The X-Men API provides information about Marvel characters who have been or currently are members of the X-Men and affiliated teams. 

Read the documentation to get started or learn how to contribute.

Want to dive in? Access all characters at:

`https://localhost:8000/api`

## Documentation

### What's in the data?

An array of character objects


|  Property |  Type | Value  |
|---|---|---|
| full name| STRING  | Character's full nam  |
| aliases  | ARRAY  |  Other names the character is known by |
| team affiliations   | ARRAY  | Current or former teams the character joined |
| first appearances | STRING | Character's first appearance in Marvel Comics
| powers | ARRAY | Character's powers and abilities

### Endpoints

The following endpoints provide access to the data.

#### All data

```js filename="endpoints.js" {3} copy

```

#### All characters

```js filename="endpoints.js" {3} copy

```


#### A specific character

```js filename="endpoints.js" {3} copy
https://xmen-api.cyclic.app/character
```


Replace `character` with the character you are searching for. 

